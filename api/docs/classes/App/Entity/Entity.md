### Entity::fromSchema  

**Description**

```php
public fromSchema (array $schema)
```

Create or update this schema from an external schema array. 

This function is ussed to update and create schema's form schema.json objects 

**Parameters**

* `(array) $schema`
: the schema to load  

**Return Values**

`$this`

> This schema


**Throws Exceptions**


`\GatewayException`
